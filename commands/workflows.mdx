## Pipeline Mode (CLI)

Chain multiple agents for complex tasks:

```bash
puzldai run "build a REST API" -P "gemini:analyze,claude:code,gemini:review"
puzldai run "task" -P "claude:plan,codex:code" -i   # Interactive: pause between steps
```

### Pipeline Actions

| Action | Description |
|--------|-------------|
| `analyze` | Analyze the task |
| `code` | Write code |
| `review` | Review and suggest improvements |
| `fix` | Fix issues |
| `test` | Write tests |
| `summarize` | Summarize content |
| `plan` | Create a plan |

## Workflow Templates

Save pipelines as reusable templates:

### Create a Workflow

```bash
# CLI
puzldai template create code-review -P "gemini:analyze,claude:code,gemini:review" -d "Code review pipeline"

# TUI
/workflows    # Opens interactive manager
```

### Run a Workflow

```bash
# TUI
/workflow code-review "my code here"

# CLI
puzldai run "task" -T code-review
puzldai run "task" -T code-review -i   # Interactive mode
```

### Manage Workflows

```bash
# CLI
puzldai template list            # List all templates
puzldai template show my-flow    # Show template details
puzldai template edit my-flow    # Edit template
puzldai template delete my-flow  # Delete template

# TUI
/workflows                       # Interactive manager
```

## Interactive Mode

Pause between pipeline steps to review:

```bash
/interactive              # Toggle in TUI
puzldai run "task" -P "..." -i   # CLI flag
```

## Autopilot Mode

Let AI generate the pipeline:

```bash
# TUI
/autopilot "build a todo app with authentication"
/planner claude                # Set planner agent
/execute                       # Toggle auto-execution

# CLI
puzldai autopilot "task"              # Generate plan only
puzldai autopilot "task" -x           # Generate and execute
puzldai autopilot "task" -p claude    # Use specific planner
```
