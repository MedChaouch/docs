## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/MedChaouch/Puzld.ai.git
    cd Puzld.ai
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Build the project">
    ```bash
    npm run build
    ```
  </Step>
  <Step title="Link globally (optional)">
    ```bash
    npm link
    ```
    Now you can run `puzld` from anywhere.
  </Step>
</Steps>

## Configuration

### API Keys

Set up your API keys for the models you want to use:

```bash
# Claude (Anthropic)
export ANTHROPIC_API_KEY="your-key"

# Gemini (Google)
export GEMINI_API_KEY="your-key"

# OpenAI Codex
export OPENAI_API_KEY="your-key"

# Ollama (local, no key needed)
# Just have Ollama running locally
```

<Tip>Add these to your `.bashrc` or `.zshrc` for persistence.</Tip>

## First Run

Start the TUI (Terminal User Interface):

```bash
puzld
```

Or run a quick prompt:

```bash
puzld run "explain what a REST API is"
```

## Basic Commands

<CardGroup cols={2}>
  <Card title="Chat" icon="message">
    Just type your message and press Enter. The AI will respond using intelligent routing.
  </Card>
  <Card title="Compare" icon="code-compare">
    ```
    /compare claude,gemini "your task"
    ```
    Compare responses from multiple models side-by-side.
  </Card>
  <Card title="Consensus" icon="users">
    ```
    /consensus claude,gemini,codex "your task"
    ```
    Models propose, vote, and synthesize a final answer.
  </Card>
  <Card title="Help" icon="circle-question">
    ```
    /help
    ```
    See all available commands.
  </Card>
</CardGroup>

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Escape` | Go back / Hide view |
| `Ctrl+E` | Expand last result |
| `Ctrl+C` | Cancel operation |
| `Tab` | Toggle Plan/Build mode |
| `↑/↓` | Navigate history |
